# 🤖 NIVA - AI Model Comparator

NAIVA is a simple and powerful web application that allows you to compare responses from multiple AI models like ChatGPT and Gemini in real-time.

---

## 🚀 Features

* 🔍 Compare multiple AI model outputs
* ⚡ Measure response latency (speed)
* 🔢 Track token usage (cost efficiency)
* 🛠 View raw backend API responses
* 📊 Side-by-side comparison dashboard

---

## 🧠 Use Cases

* AI model evaluation
* Performance comparison
* Debugging API responses
* Learning how different models behave

---

## 🛠 Tech Stack

* Python
* Streamlit
* OpenAI API (ChatGPT)
* Google Gemini API


---

## 📌 How to Use

1. Enter your prompt
2. Click **Run Comparison**
3. View responses from ChatGPT and Gemini
4. Analyze latency and token usage
5. Expand debug section for raw API data
